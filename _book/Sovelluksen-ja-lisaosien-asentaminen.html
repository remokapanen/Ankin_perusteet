Ankin perusteet

Chapter 2 Sovelluksen ja lisäosien asentaminen

2.1 Lataa Anki koneelle

Anki on ilmainen ohjelma, joka on saatavilla Windowsille, Macille ja Linuxille. Lataa ohjelma osoitteesta https://apps.ankiweb.net/.

  • Kun olut ladannut ja avannut ohjelman, sinun tulee valita käyttäjä. Voit jättää User 1:n omaksesi tai muuttaa sen nimeä, tällä ei ole sinänsä merkitystä tässä vaiheessa. Paina tämän jälkeen Open.

2.2 Lisäosat

Lisäosat avaavat Ankin oikeat kyvykkyydet. Olen koonnut tiedoston, jossa on kaikki ainakin nyt alkuun suosittelemani lisäosat.

  • Mene Tools -> Add-ons -> Install from file

Lisäosia voi myös ladata yksitellen koodien avulla: Tools -> Add-ons -> Get Add-ons -> Code-kohtaan alla olevia koodeja yksi kerrallaan. Yllä olevassa zip-tiedostossa on seuraavat lisäosat:

  • Lääketiedepainotteiset
    • 1957538407 (AnkiCollab)
    • 952691989 (AnKing Note Types Easy Customization)
    • 1052724801 (Bettersearch)
    • 1374772155 (Image Occlusion Enhanced)
    • 272311064 (Closet For Anki)
  • Kielipainotteiset
    • 2055492159 (AnkiConnect)
  • Yleiset
    • 1771074083 (Review heatmap)
    • 1247171202 (Study Time Stats)
    • 291119185 (Batch editing)
    • 138501288 (Search and Replace Tags)
    • 354407385 (Opening the same window multiple times)
    • 24411424 (Customize Keyboard Shortcuts)
    • 1362209126 (Quizlet Importer)
    • 759844606 (FSRS Helper)
    • 1046608507 (Speed Focus Mode)
    • 1020366288 (Edit field during review)

Aina lisäosien lisäämisen jälkeen on syytä sulkea Anki ja avata se uudestaan.
